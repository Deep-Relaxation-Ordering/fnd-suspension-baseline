## What changed from `pilot-v0.1`

1. **Rayleigh convection gate.**
   [`src/convection.py`](../../src/convection.py) derives water's
   signed thermal expansion coefficient from the same Tanaka density
   fit used elsewhere, computes Rayleigh numbers, and exposes a
   `convection_flag` side channel through `regime_map`. The committed
   cache uses `delta_T_assumed = 0.0 K`; notebooks use
   `DEFAULT_EXPERIMENTAL_DELTA_T_K = 0.1 K` for overlays.
2. **Hydrodynamic/material radius split.**
   `ParticleGeometry` separates `r_material_m` (buoyant mass) from
   `r_hydro_m = r_material_m + delta_shell_m` (drag and diffusion).
   Legacy scalar-radius calls remain wrappers, and the cache reader
   accepts v0.1, Phase-11, and v0.2 CSV headers.
3. **Log-normal polydispersity smearing.**
   [`src/polydispersity.py`](../../src/polydispersity.py) turns the
   deterministic monodisperse cache into label probabilities over
   `sigma_geom = {1.05, 1.10, 1.20, 1.40, 1.60}`. Notebook 05 ships
   deliverable 6: a room-temperature design table plus three figures.
