## Validation

The schema drift guard is
[`../../tests/test_data_schemas.py`](../../tests/test_data_schemas.py).
It checks that every CSV in this directory has a schema, every schema
points at an existing CSV, headers match exactly, row counts match, and
scalar values parse according to their declared Frictionless field
types.

The schemas are additive FAIR metadata. They do not change any cached
physics values and do not require notebook regeneration.
