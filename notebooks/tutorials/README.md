# Tutorial notebooks

Visitor-facing tutorials will live here as jupytext `.py` notebooks.
They are accessibility surfaces, not release artefacts.

Current status: no tutorials are ready at `pilot-v0.4`; the proposed
v0.5 learning path is tracked in
[`../../docs/tutorial-roadmap.md`](../../docs/tutorial-roadmap.md).

Tutorial rules live in
[`../../docs/conventions.md`](../../docs/conventions.md) under
"Tutorial notebooks". Ready tutorials must carry a stable `TUT-*`
identifier, release tag, canonical input list, smoke-test command,
and citation / reuse links back to `CITATION.cff`, `codemeta.json`,
`LICENCE`, and the license metadata in `pyproject.toml`.
